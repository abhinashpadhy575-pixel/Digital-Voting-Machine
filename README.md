# Digital Voting Machine  A console-based voting system developed in C++. The project allows users to cast votes, prevents duplicate voting through Voter ID verification, displays election results, calculates total votes, determines the winner, and handles tie situations between candidates. 
## Features  - Cast votes for candidates - Voter ID verification - Prevent duplicate voting - Display vote counts - Show total votes cast - Winner detection - Tie handling - Menu-driven interface  
## Technologies Used  - C++ - Functions - Arrays - Loops - Conditional Statements - Switch Case - Pass-by-Reference  
## Learning Objectives  This project was built to practice: - Problem-solving in C++ - Function design - Array manipulation - Input validation - Basic algorithm implementation  
## Future Improvements  - Admin authentication - Dynamic candidate names - File handling for persistent storage - Candidate management using structures (`struct`) - Election reports and statistics
